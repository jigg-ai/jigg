# Jigg.AI

Build-in-public project: real things built with AI tools, documented honestly —
tested, timed, and reproducible. The site is built with the tools it reviews.

## Read these first
- `CONTEXT.md` — what Jigg.AI is and why (the concept). Source of truth.
- `PROCESS.md` — the repeatable build pipeline.
- `STYLE.md` — house voice for published logs.
- `site/BRIEF.md` — the brief for building the site itself (build #1).

## Layout
- `builds/` — one folder per build. Copy `_template/` to start a new one.
- `site/` — the Astro site.

The repo is the durable record. Chats are scratchpads; the source of truth lives here.
Keep `CONTEXT.md` mirrored into Claude Project knowledge; when it changes here, re-upload.
