# Affiliate disclosure

Source page: https://jigg.ai/affiliate-disclosure/

Some links to tools on Jigg.AI are affiliate links. Here's exactly what that means — and
what it never changes.

Some of the links to tools on this site are affiliate links. If you sign up or buy
through one, Jigg.AI may earn a commission — at no extra cost to you.

**This is how Jigg.AI makes money:** affiliate commissions from the tools it features.
The site earns its keep from the tools it uses to prove what's possible.

Here's what that changes about the write-ups: nothing.

## The rules I hold myself to

- Tools are chosen on merit — worth your time, accessible, and actually useful for
  building something real. An affiliate program never buys coverage.
- Every build is tested and written up honestly, including what broke and what didn't
  work. The affiliate relationship doesn't soften the verdict.
- Affiliate links are always disclosed, and always placed *after* the proof — once you've
  seen what was actually built — never before it.
- If a tool isn't worth it, the post says so, affiliate program or not.

The commissions are what let Jigg.AI keep building in the open. But the whole project
only works if you can trust the write-ups — so the proof always comes first, and the link
always comes second.

## The bot you can chat with

The support chatbot in the corner of this site is built with Botpress. When Jigg.AI links
to Botpress, that link may be an affiliate link — so the widget is both the honest proof
and, indirectly, what the link points to. The same rule holds: the proof comes first (a
working bot you can actually try), and the "build one yourself" link comes after.
